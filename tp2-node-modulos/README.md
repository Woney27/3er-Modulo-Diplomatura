TP2 - Node.js módulos nativos

Este repositorio contiene ejemplos de uso de módulos nativos de Node.js:

- path
- os
- fs
- events
- http

Requisitos
Node.js instalado

Ejecución de ejemplos

PATH
node path/pathExample.mjs

OS
node os/osExample.mjs

FS
node fs/fsWriteRead.mjs

EVENTS
node events/eventExample.mjs

HTTP
node http/server.mjs
